# note

[Hexo](https://hexo.io/)

## Quick Start

### Installation

```bash
$ npm install
```

### Create a new post

``` bash
$ npx hexo new "My New Post"
```

More info: [Writing](https://hexo.io/docs/writing.html)

### Run server

``` bash
$ npx hexo server
```

More info: [Server](https://hexo.io/docs/server.html)

### Generate static files

``` bash
$ npx hexo generate
```

More info: [Generating](https://hexo.io/docs/generating.html)

### Clean

``` bash
$ npx hexo clean
```
