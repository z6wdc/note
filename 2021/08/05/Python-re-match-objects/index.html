Python re Match Objects

Match Objects

Match.group

可以利用 group() 來列印出匹配的值

index 預設為零

如果正規表達式有設定 (?P<name>...) 的 syntax

可以利用 name 來指定匹配到的值

沒有匹配到的話，就會回傳 None

Match.lastindex

使用 group() 的時候

如果代入負數或是大於匹配的結果

會發生 IndexError

可以利用 lastindex 先找出最大的 index

Match.lastgroup

如果正規表達式有設定 (?P<name>...) 的 syntax

可以用 lastgroup 來找最後一筆匹配的值的 name

Match.groups

可以利用 groups() 來一次列印出匹配的值

如果沒有成功匹配就會印出 None

Match.groupdict

可以一次列印出設定的 name 與對應的值

如果沒有成功匹配就會印出 None

一樣可以設定預設值
