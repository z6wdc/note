value_counts

normalize

Python mutable immutable

Python 的 object 有分成

mutable 跟 immutable 兩種

mutable object 是指

object 的值可以被更改

immutable 則反之

當 object 被 create 之後，值就不能被更改

mutable example

例如：list, dictionary

list example

Output

immutable example

例如：numbers, string, tuple

但 immutable object 還需要注意

值不能被更改的意思

numbers example

Output

可以看到 a 的值被更改了

但如果用 id() 把 a 的 address 印出來的話

就會發現 a 的 address 已經改變了

因為 a 是 immutable object

所以對 a 指定新的值的時候

就會被指去新的 address

雖然值更換了，但也不是原本的 object 了

tuple example

還有另外一個關於 tuple 的例子

雖然 tuple 是 immutable object

但如果 tuple 裡面包含 mutable object

譬如說 list

那在 tuple 裡面的 list 是可以更改值的

Python random module

Python 有 random 這個 module
除了可以產生亂數外，還可以隨機挑選資料

How to import

Functions for integers

randint

randint 會隨機從 0 <= n <= 100 的數字隨機挑選一個整數

Functions for sequences

choices

choices 會隨機從指定的 list 中重複取出你要的數目

要注意的是指定數目的時候需要寫出 keyword k

像是例子中 k=3 就會取出三個

不指定 k 的話，預設值為 1

sample

sample 會隨機從指定的 list 中取出你要的數目

而且數字不重複

但要取出的數目不能大於 list 的 length

不然就會發生下面的

ValueError: Sample larger than population or is negative
