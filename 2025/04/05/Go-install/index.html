安裝方法整理

flowchart TD
    A[安裝 Go] --> B{哪種方式安裝？}

    B --> C1[官方 .pkg 安裝]
    B --> C2[go install golang.org/dl/goX.Y.Z@latest]

    %% .pkg 安裝流程
    C1 --> D1[/usr/local/go 被建立/]
    D1 --> E1[/etc/paths.d/go 被寫入/]
    E1 --> F1[PATH 自動包含
/usr/local/go/bin]
    F1 --> G1[go 指令可用 ]

    %% go install 安裝流程
    C2 --> D2[goX.Y.Z 安裝到 ~/go/bin]
    D2 --> E2{~/go/bin 是否在 PATH？}

    E2 -- 否 --> F2[需要手動加入]
    F2 --> G2[在 ~/.zshrc 或 ~/.zprofile
新增 PATH]
    G2 --> H2[執行 source ~/.zshrc
或重新開 Terminal]
    H2 --> I2[goX.Y.Z download指令可用]

    E2 -- 是 --> I2
